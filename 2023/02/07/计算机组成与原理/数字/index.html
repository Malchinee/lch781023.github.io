数字

无符号数和有符号数

无符号数

只有数值部分，将其变换成二进制，用寄存器或存储器用二进制保存即可

寄存器长度直接反映了无符号数的表示范围

有符号数

有正负号的数字。包含了数值部分和符号部分

机器数与真值

假设规定小数点位置在符号位的后一个

假设小数点位置在数值为之后

如果小数点位置像第一种，被称为小数定点机；像第二种一样，就是整数定点机

如果把符号数字化，就是原码表示

原码

整数：

  • 定义：用n表示数值部分的位数，表示一个存储单元最多存储几位；x为真值

  • 举例

如 x = + 1110， ；X = - 1110 ，

可见如果是正数，就在前面加0逗号，存储；如果是负数，在前面加1逗号，存储。

因此原码是带符号的绝对值表示

小数：

  • 定义：

  • 举例

假设小数点在符号位的后面

如x = + 0.1101 ，但是这两个0的含义不一样，前一个表示整数为0，后一个表示符号位为正数

x = - 1.1101，

x = 0的原码有4个，正小数假设为+0.0000，它的原码为0.0000；负小数假设为-0.0000，它的原码为1.0000；同时整数为0,0000和1,0000

原码的特点：简单、直观。但是计算不容易

补码

一个负数加上模可以得该负数的补数，一个正数和一个负互为补数时，它们绝对值之和即为模数

减去一个数可以看成加上一个数的补数

定义：

  • 整数

    注意整数0的补码正数和负数是相等的

  • 小数

    注意计算机存储是有限长度的，即正小数补码的值不一定和真值相等

举例：

  • 整数

  • 小数

求补码的快捷方式：真值为负数时，原码除符号位外，每位取反，末位加1

0的补码只有一个，是0

-1.0000有补码为 1.0000，但是无原码

反码

定义：

  • 整数

    负数的原码除符号位外取反加1得到补码，其中这个1不加了，就是反码

  • 小数

正数反码是本身，负数反码除符号位外每位取反

0 的反码有两个0. 0000和1.1111

三种机器数小结：

  • 最高位为符号位，书写用“，”或“.”将数值部分和符号位隔开
  • 对于正数：原码＝补码=反码
  • 对于负数：最高位符号位为1，其真值部分，原码等于真值；反码等于原码取反；补码等于反码＋1

浮点数的规格化形式：

举例：

当m = 4，n = 10， r = 2时，所表示的范围

  • 最大正数：数符为0，尾数最大，阶符为0，阶码最大。

  • 最小正数：数符为0，尾数最小，阶符为0，阶码最小。

  • 最大负数：数符为1，尾数最小，阶符为1，阶码最小。

  • 最小负数：数符为1，尾数最大，阶符为0，阶码最大.

机器零：

  • 当浮点数尾数为0时，不论阶码为何值，按机器零处理

  • 当浮点数阶码等于或小于它所表示的最小数时，不论尾数为何值，按机器零处理
